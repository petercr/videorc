# Videorc 0.9.0 Beta 1

Videorc 0.9.0 beta 1 is the first signed macOS beta candidate for the local recording studio.

## Scope

- Signed and notarized macOS DMG.
- Local screen, camera, and microphone recording.
- Native detached preview window.
- Source and layout controls.
- Bundled backend, native preview helper, and FFmpeg.
- Authenticated download through Videorc Web.

## Acceptance Gate

This beta is releasable only after the clean-machine macOS acceptance pass is
recorded under `docs/acceptance/`.
